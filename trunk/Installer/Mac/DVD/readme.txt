Virtual Planet Atlas
version 1.0
Free software for the planet observation or survey
For Mac OS X

This software, realized by Patrick Chevalley and Christian Legrand can visualize the planets aspect for every date and hour. It permits also to study the formations with unique database compiled by Christian Legrand.
Version for Linux and Windows are also available.

INSTALLATION:

To install this software you need to click on the files in the DVD, do not copy this files to Applications!

Install first the file virtualplanet_pro-1.0-macosx-i386.pkg with the software and the basic textures.

Then you can install virtualplanet-extra-1.0-macosx.pkg to add the high resolution textures.
