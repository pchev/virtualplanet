Virtual Planet Atlas
version 1.0
Free software for the planet observation or survey
For Windows XP/Vista/7/8/10

This software, realized by Patrick Chevalley and Christian Legrand can visualize the planets aspect for every date and hour. It permits also to study the formations with unique database compiled by Christian Legrand.
Version for Linux and Mac OS X are also available.
