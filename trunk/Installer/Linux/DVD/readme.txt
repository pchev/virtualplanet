Virtual Planet Atlas
version 2.0
Free software for the planet observation or survey
For Linux

This software, realized by Patrick Chevalley and Christian Legrand can visualize the planets aspect for every date and hour. It permits also to study the formations with unique database compiled by Christian Legrand.
Version for Windows and Mac OS X are also available.

INSTALLATION:

Depending on your Linux system you must use the files in the DEB or RPM directory.

Install first the file virtualplanet corresponding to your processor architecture and virtualplanet_data with the basic textures.

Then you can install virtualplanet-extra to add the high resolution textures.
